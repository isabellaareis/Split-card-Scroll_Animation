# Scroll OVA

Este projeto é uma "cópia" do flipping Cards do site Redo Media

Técnologias ultilizadas:

- HTML
- CSS
- Java Script
- GSAP

  # Como executar o projeto localmente

1 - Clone este repositório

2 - Abra o projeto no VSCODE (ou no editor de sua preferência).

3 - Inicie a depuração

4 - Acesse o projeto no navegador (geralmente com o endereço http://127.0.0.1:5500/Index.html).


  # ⚠️ Observação

Este projeto foi desenvolvido como parte de um **estudo prático**, então não esta com muito contéudo e completo.
